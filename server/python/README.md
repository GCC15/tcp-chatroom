# tcp-chatroom/server/python

A Python implementation of the chatroom server.

*Under development.*

# Requirements

- A host with a public IP address

- Python 3

# Usage

- Configure

  [config.json](config.json)

- Check server status

  ```
  python scrpd.py status
  ```

- Start the server

  ```
  python scrpd.py start
  ```

- Stop the server

  ```
  python scrpd.py stop
  ```
